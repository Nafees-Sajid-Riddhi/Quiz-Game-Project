# Quiz-Game-Project

Project completed for our Data Structures and Algorithms course, built fully in Java using IntelliJ IDEA.

The program provides a simple quiz game with a graphical user interface (GUI). Users log in and answer multiple-choice questions to earn points.

## How to Run
1. Download or clone the repository.
2. Open the `QuizGameProject` folder in IntelliJ IDEA.
3. Navigate to the `/src` folder.
4. Run the `QuizLogin.java` file.
5. When the program starts, the GUI login window will appear.
6. Log in using the password '12345'.
7. The quiz will begin. Select the correct answers to score points. Wrong answers do not add marks.
